🎮 CÓMO USAR
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. CREAR CLIENTE
   👥 Clientes → + Nuevo Cliente → Llenar formulario → Guardar

2. REGISTRAR VENTA CON CLIENTE
   💳 Ventas → Seleccionar producto → Seleccionar cliente → Registrar

3. GENERAR FACTURA
   📄 Facturas → + Nueva Factura → Seleccionar cliente → Descargar PDF

4. CERRAR MES
   📊 Cierre Mes → Seleccionar mes → Ver Resumen → Cerrar Mes
